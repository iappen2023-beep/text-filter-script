# text-filter-script
Python script that filters lines from text file by keyword

## What it does
Reads input.txt, identifies lines containing "PCR", and writes each with its original line number to output.txt. Also prints results to the terminal.

## How to run
Place input.txt in the same folder as script.pythen run:
    python script.py

## Files

- script.py: the script
- input.txt: sample input with life sciences and biological style lab notes
- output.txt: produced by running the script
